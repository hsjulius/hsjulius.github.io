    • Brush + Filter

    In this project I created 3 different brush masks to simulate Photoshop-like color brushes. The brushes included a constant, linear, quadratic, and spray brush. Each of these describes the intensity of the color placed onto the canvas. For example, a pixel x distance away from the center in a linear brush would have an intensity directly proportional to the distance x. I also created a smudge brush to blend the color being placed with the color currently on the canvas. Then I created a mode to fix alpha blending, so that if a brush passes over the same location within a single stroke that the opacity does not increase past the expected opacity of the brush. As part of the project I also implemented 2D filters with convolution.
